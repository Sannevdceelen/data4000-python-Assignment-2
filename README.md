# DATA 4000 – Python with AI Assignment 2

This repository contains solutions for **Assignment 2** of the course **DATA 4000: Python with AI**.
The programs demonstrate core Python concepts including variables, user input,
conditional statements, functions, and basic input validation.

All code is written in **Python 3** and intended to be run from the command line using **Visual Studio Code**.

---

## Problem 1: Customer Discount Eligibility
File: `problem1_customer_discount.py`

Prompts the user for a purchase amount and membership status, determines the
appropriate discount based on the given rules, and prints the final price after
the discount is applied.

Assumptions:
- Purchase amount is a numeric value.
- Membership status is entered as `yes` or `no`.
- Discount rules follow the assignment specifications.

---

## Problem 2: Employee Performance Bonus
File: `problem2_employee_bonus.py`

Evaluates an employee’s performance rating and years of service to determine
bonus eligibility and prints the corresponding bonus message.

Assumptions:
- Performance ratings and years of service are numeric inputs.
- Invalid inputs are handled appropriately.

---

## Problem 3: Loan Risk Classification
File: `problem3_loan_risk.py`

Classifies a loan applicant as **Low Risk**, **Medium Risk**, or **High Risk**
based on credit score and annual income using conditional logic.

Assumptions:
- Credit score is a positive integer.
- Annual income is a positive numeric value.
- Classification follows the rules provided in the assignment.
